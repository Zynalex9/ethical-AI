# Benchmark Datasets Directory

This folder contains benchmark CSV files used by the `/datasets/project/{project_id}/load-benchmark` API.

Required filenames:

1. `compas-scores-raw.csv`
2. `adult.csv`
3. `german_credit_data.csv`
4. `bank_marketing.csv`
5. `diabetes_130_us_hospitals.csv`

The backend currently supports these dataset keys:

- `compas`
- `adult_income`
- `german_credit`
- `bank_marketing`
- `diabetes_readmission`

You can replace these starter files with full benchmark datasets from public sources while keeping the same filenames.
